Before you start anything, check:
> Is nodemon installed? 
- npm install -g nodemon

> Is nodejs installed? 
- Install it yourself somewhere


To start localhost:
- open the project in Terminal
- run 'nodemon server.js'
- open http://localhost:3000/index.html
You're done.
